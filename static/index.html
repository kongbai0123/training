UI Ready / Backend Pending

How to use LabelMe in this platform

Phase 1 UI / Phase 2 Sync
  1. 先到 Dataset 匯入圖片，圖片會放在下方的 Images folder。
  2. 在本機開啟 LabelMe，選擇 Open Dir，指向 Images folder。
  3. 在 LabelMe 內完成 polygon / rectangle 標註，並把 JSON 存到 LabelMe JSON folder。
  4. 回到本平台按 Sync JSON。目前 Phase 1 尚未接後端，所以按鈕先停用。
  5. Phase 2 會解析 JSON、顯示錯誤清單、預覽標註，並轉成 YOLO / COCO / Mask 格式。
